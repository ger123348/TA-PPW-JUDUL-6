# TA-PPW-JUDUL-6

Gerhana Malik Ibrahim (2315061032)

web yang menampilkan informasi cuaca menggunakan data dummy JSON untuk beberapa kota di Indonesia. Fitur utama meliputi tampilan cuaca saat ini (temperature, humidity, wind speed, condition & icon), forecast 5 hari, pencarian kota dengan auto-suggest, penyimpanan kota favorit, toggle Celsius/Fahrenheit, mode Dark/Light, loading indicator, refresh data, serta pembaruan otomatis setiap 5 menit. Aplikasi ini dibuat responsif sehingga nyaman digunakan pada perangkat mobile maupun desktop, serta menyediakan pesan error ketika kota tidak ditemukan.
